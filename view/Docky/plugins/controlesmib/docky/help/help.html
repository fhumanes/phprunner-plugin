WELCOME TO THE DOCKY PLUGIN PAGE


This page contains a fully simple explanation about a PHPRunner plugin. The intention of making this object is to offer a way to preview any standard document for PHPRunner and also to contribute to make Xlinesoft Marketplace bigger. This plugin is written entirely in PHP and CSS, using JAVASCRIPT JQUERY and it's based on the fantastic concepts shown in this article.

It's the eleventh plugin I've made. I hope you to enjoy programming using this little tool, it will save you hours in front of your screen. Feel free to send your opinion. You can view more MIB Plugins for PHPRunner soon.




COMPATIBILITY AND VERSIONS


This plugin was made only for PHPRunner.
Tested in version 9.8, with bootstrap and not bootstrap layouts.
It uses the standard structure for all plugins, so you could use in any other version.


It's only necesary one field to store the document. Blob type is mandatory. Optionally, you can set a field to store the name.

INITIAL PARAMETERS

As any other plugin, you have to initialize it providing certain parameters to let the control to function properly as your needs. Many of them are self-explanatory. Here's the list of parameters for version 1.0:

MAIN PROPERTIES:
"fielddocname": Field to store document's name
"title": Header caption
"defaultdocname": Default document's name if it's not set

When you initialize the plugin you'll see examples commented for all parameters. It's very simple to use!

Be sure to have write permissions on plugins/controlesmib/docky/content/files (Docky needs to create and delete files there). Check daily the space in plugins/controlesmib/docky/content/files in order to control if Docky is generating waste. Files generated by Docky has to be reachable by Google Docs Viewer, so it does not work on local environments.
